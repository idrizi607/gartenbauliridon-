# 🌿 Garten Bau Liridon: Luxury Landscape Architecture

A high-end, responsive landing page developed for a premier landscaping firm specializing in modern estate design and green space maintenance. This project bridges the gap between traditional craftsmanship and modern digital branding.

---

## 💎 Design Philosophy
The UI was built around a "Modern Luxury" aesthetic. By utilizing a high-contrast palette of **Onyx Black, Pure Gold, and Minimalist White**, the interface reflects the precision and premium nature of the physical landscaping work.

- [/] **Service Modules:** Developing a dynamic grid for Private Estates, Modern Gardens, and Maintenance.

## ✨ Technical Highlights
* **Performance-First:** Written in clean, semantic HTML5 and CSS3 to ensure sub-1-second load times.
* **Fluid Responsiveness:** Mobile-first architecture that scales perfectly from smartphones to 4K monitors.
* **Motion Design:** Subtle micro-interactions used to guide the user's eye toward Call-to-Actions (CTAs).

## 🧰 Tech Stack
* **Core:** HTML5, CSS3, JavaScript (ES6+)
* **Animation:** CSS Keyframes & planned GSAP ScrollTrigger
* **Deployment:** GitHub Pages / Professional Version Control
